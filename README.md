# Credit Card Check App
This app validates credit card input.

# How to run?.
1. Clone the repository

  ```bash
  $> cd your-folder
  $> node install
  $> node server.js
  ```
  
2. Open a broswer and visit
  ```bash
  localhost:3000
  ```
